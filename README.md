https://play.gikopoi.com

Like [iccanobif/gikopoi2](https://github.com/iccanobif/gikopoi2) but with bees
and misc hacks. 

to start server, run:

``$ ./compile.sh``
``$ ./run.sh``

for local debugging:
``$ yarn dev``

to enable automatic version bump run:

``$ git config core.hooksPath .githooks``
