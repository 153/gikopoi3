Spells

  • #warp / #rula -- change room
  • #list -- pull up user list (useful for ignoring troublemakers)
  • #afk / #ghost / afk -- mark yourself AFK
  • #bee -- beecome a bee
  • /me message / #me message -- IRC style "/me" command
  • #d6 / #3d20 / etc -- roll x many dice with y sides, capital D to show independent rolls
  • #spy message -- broadcast global message anonymously
  • !help -- use giko.py bot commands if giko.py is present
